# bbtime

Network time synchronisation for **BlackBerry OS 4.5+** (classic RIM Java).

Fetches authoritative time over **NTP, HTTP, HTTPS, or UDP (RFC 868 TIME)**,
measures the offset from the device clock, and lets you sync with a single key
press. Optionally runs a once-a-day background check.
